Semantic HTML Reference

Semantic Tags

Tag Name Description
<header> Header A container usually for navigational link or introductory content containing headings.
<main> Main Describes where the main information of webpage is located.
<nav> Nav Used to align a block of navigation links.
<footer> Footer Contains author information, site map, copywright information. etc
<article> Article Holds content that makes sense on its own. ex. text, image, audio
<section> Section Defines elements in an articles, ex, chapters and headings.
<aside> Aside Used to mark additional information that can enhance another element but isn't required to understand the main content.
<figure> Figure Used to contain media, ex: image, illustration..., which is referenced in the flow of the document.
<figcaption> Figcaption Used to describe media inside <figure>
<audio> Audio Used to embed audio content into a document
<video> Video Embeds videos on the site
<embed> Embed Embeds any media content from an external source. Self closing tag.

Semantic Attributes

Attribute Name Description
